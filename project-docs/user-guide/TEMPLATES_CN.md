# 📋 笔记模板

使用可重用模板创建笔记，带动态占位符替换。

## 概述

模板允许您快速创建具有预定义结构和内容的笔记。非常适合 recurring 笔记类型，如会议笔记、每日日志、项目计划等。

## 如何使用模板

### 1. 创建模板文件

模板存储在 `data/_templates/` 文件夹中，为 Markdown 文件。如果该文件夹不存在，您需要创建它：

```
data/
└── _templates/
    ├── meeting-notes.md
    ├── daily-journal.md
    └── project-plan.md
```

### 2. 访问模板

点击**"新建"**按钮（或任意文件夹上的 **+**），选择**"从模板新建"**：

1. 从下拉菜单选择模板
2. 为新笔记输入名称
3. 点击"创建笔记"

模板将被复制，所有占位符自动替换！

### 3. 使用占位符

模板支持动态占位符，创建笔记时会自动替换：

| 占位符 | 描述 | 示例 |
|------------|-------------|---------|
| `{{date}}` | 当前日期 | `2025-11-26` |
| `{{time}}` | 当前时间 | `14:30:45` |
| `{{datetime}}` | 当前日期和时间 | `2025-11-26 14:30:45` |
| `{{timestamp}}` | Unix 时间戳 | `1732632645` |
| `{{year}}` | 当前年份 | `2025` |
| `{{month}}` | 当前月份 | `11` |
| `{{day}}` | 当前日期 | `26` |
| `{{title}}` | 笔记名称（不含 .md） | `Weekly Meeting` |
| `{{folder}}` | 父文件夹名称 | `Projects` |

### 示例模板

```markdown
---
tags: [meeting]
date: {{date}}
---

# 会议笔记 - {{title}}

**日期：** {{datetime}}
**参与者：**
-

## 议程
-

## 讨论


## 待办事项
- [ ]

## 后续步骤


```

当您从此模板创建名为"团队同步"的笔记时，它将变为：

```markdown
---
tags: [meeting]
date: 2025-11-26
---

# 会议笔记 - 团队同步

**日期：** 2025-11-26 14:30:45
**参与者：**
-

## 议程
-

## 讨论


## 待办事项
- [ ]

## 后续步骤


```

## 示例模板

我们在 `project-docs/templates/` 中提供三个示例模板，您可以复制到 `data/_templates/` 文件夹：

1. **meeting-notes.md**——结构化的会议笔记，含议程、讨论和待办事项
2. **daily-journal.md**——每日日志，含晨间目标和晚间反思
3. **project-plan.md**——项目规划模板，含目标、时间线和状态跟踪

### 使用示例模板

**选项 1：手动复制**
```bash
cp project-docs/templates/*.md data/_templates/
```

**选项 2：创建您自己的**
1. 在 `data/_templates/` 中创建 `.md` 文件
2. 添加内容和占位符
3. 保存即可使用！

## 提示

- ✅ 模板可以包含 YAML frontmatter 用于标签和元数据
- ✅ 使用描述性模板名称（它们会出现在下拉菜单中）
- ✅ 模板可在任意文件夹中使用——上下文保留
- ✅ 您可以随时编辑模板——更改仅应用于新笔记
- ✅ 将模板与标签结合，实现强大的组织功能
