# 🔗 公开分享

无需登录即可公开分享笔记供他人查看。

---

## 🎯 核心功能

- ✅ **零门槛访问** — 接收者无需注册账号
- ✅ **主题保留** — 分享时使用创建链接时的主题
- ✅ **图片嵌入** — 所有图片包含在分享视图中
- ✅ **代码高亮** — 语法高亮正常工作
- ✅ **一键复制** — 代码块提供复制按钮
- ✅ **Math & Mermaid** — LaTeX 公式和图表正常渲染
- ✅ **二维码** — 方便手机扫描访问
- ✅ **永不过期** — 链接直到撤销前一直有效
- ✅ **只读访问** — 查看者无法编辑

---

## 📤 如何分享笔记

**步骤：**

1. 打开要分享的笔记
2. 点击工具栏中的 **Share**（分享）按钮 📤
3. 点击 **Create Share Link**（创建分享链接）
4. 复制生成的 URL，或点击 **Show QR Code** 显示二维码
5. 将链接/二维码发送给他人

**接收者访问：**  
直接在浏览器中打开链接即可查看笔记内容。

---

## 🚫 撤销分享

要停止分享某笔记：

1. 打开该笔记
2. 点击 **Share**（分享）按钮
3. 点击 **Revoke Link**（撤销链接）

**效果：**

- 旧链接立即失效
- 访问时将返回 404 错误
- 无法恢复（需重新创建分享）

---

## 🔍 视觉指示器

侧边栏中，已分享的笔记旁会显示**分享图标** 🔗，便于快速识别。

分享对话框会显示：

- 当前分享状态（已分享 / 未分享）
- 分享链接 URL
- 创建时间
- 使用的主题

## 🔧 技术细节

### Token 存储

分享令牌（token）存储在数据目录下的 `.share-tokens.json` 文件中：

```json
{
  "LRFEo86oSVeJ3Gju": {
    "path": "folder/note.md",
    "theme": "dracula",
    "created": "2026-01-15T10:30:00+00:00"
  }
}
```

**限制：**

- 每条笔记只能有一个分享 token
- 对已分享笔记再次创建链接时，返回现有 token
- token 为 16 位随机字符串，具有足够熵值

---

### 安全性说明

| 特性 | 说明 |
|------|------|
| **Token 强度** | 16 位随机字符串，难以猜测 |
| **访问控制** | 只有精确 token URL 才能访问 |
| **撤销机制** | 删除 token 立即失效 |
| **权限** | 只读（无法编辑或修改） |
| **持久化** | token 文件需备份（与笔记数据一起） |

**注意：**  
分享功能适合轻量级公开分享场景。如需更复杂的访问控制（如密码保护、过期时间、访问统计），建议使用其他方案。

## 🔗 分享链接格式

```
https://your-domain.com/share/{token}

# 示例
https://gonote.example.com/share/LRFEo86oSVeJ3Gju
```

公开端点 `GET /share/{token}` 返回独立 HTML 页面，加载时：

1. 从 token 查找笔记路径
2. 读取笔记内容
3. 应用创建时的主题样式
4. 渲染完整页面（含 QR 码可选）

## 🔄 常见问题

### Q：分享的笔记可以编辑吗？

A：不可以。分享链接为只读访问，接收者只能查看，无法修改。

---

### Q：分享链接会过期吗？

A：默认永不过期，直到您在应用中撤销分享。建议定期审查已分享笔记。

---

### Q：可以批量分享多篇笔记吗？

A：目前每篇笔记单独分享。未来版本可能支持集合分享。

---

### Q：分享的图片可以访问吗？

A：可以。所有笔记中的图片通过媒体 API 提供，分享视图中直接嵌入显示（以 Base64 或相对路径方式）。

---

### Q：如何知道哪些笔记已分享？

A：侧边栏中已分享笔记旁有 🔗 图标。点击分享按钮可查看状态。

---

## 📊 管理已分享笔记

2. 底部会显示当前已分享笔记数量
3. 或直接查看 `.share-tokens.json` 文件（技术用户）
