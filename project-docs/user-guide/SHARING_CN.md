# 🔗 公开分享

无需登录即可公开分享笔记供他人查看。

## 工作原理

1. 打开你想要分享的笔记
2. 点击工具栏中的 **Share**（分享）按钮
3. 点击 **Create Share Link**（创建分享链接）
4. 复制生成的 URL，或点击 **Show QR Code**（显示二维码）方便手机扫描

接收者可以在浏览器中查看笔记——无需注册账号。

## 撤销访问

要停止分享笔记：
1. 打开该笔记
2. 点击 **Share**（分享）按钮
3. 点击 **Revoke Link**（撤销链接）

旧链接将立即失效。

## 功能特性

- **主题保留**——分享的笔记会显示创建链接时激活的主题
- **图片嵌入**——所有图片都包含在分享视图中
- **代码高亮**——代码语法高亮功能在分享视图中正常工作
- **复制按钮**——代码块提供一键复制到剪贴板功能
- **MathJax 与 Mermaid**——数学公式和图表可正常渲染
- **二维码**——生成二维码，方便手机分享
- **永不过期**——链接在撤销前一直有效

## 视觉指示器

- 侧边栏中已分享的笔记旁会显示**分享图标**
- 分享对话框会显示当前的分享状态

## 技术细节

### Token 存储

分享令牌（token）存储在数据目录下的 `.share-tokens.json` 文件中：

```json
{
  "LRFEo86oSVeJ3Gju": {
    "path": "folder/note.md",
    "theme": "dracula",
    "created": "2026-01-15T10:30:00+00:00"
  }
}
```

每条笔记只能有一个分享 token。对已分享的笔记再次创建链接时，将返回现有的 token。

### 安全性

- Token 是随机生成的 16 位字符串
- 只有精确的 token URL 才能授予访问权限
- 撤销操作会永久删除该 token
- 分享的笔记为只读状态（查看者无法编辑）
